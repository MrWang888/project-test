Node:

安装express：

npm i express-generator -g

生成一个项目:

express --view=ejs "项目的名字"

cd 项目文件夹

cnpm i     //装依赖

npm start    //启动项目

Vue常用指令:

安装脚手架:

cnpm i vue-cli -g 只需要安装一次

构建一个项目:

vue init webpack


装依赖:

cnpm i


安装json-server:

cnpm i json-server -g

安装axios模块:

cnpm i axios --save


安装路由:

cnpm i vue-router --save

安装vue-jsonp:

cnpm i vue-jsonp --save

运行开发环境:

npm run dev

安装sass:

cnpm i node-sass sass-loader --save

安装swiper:

cnpm i swiper --save

安装mint-ui库:

cnpm i mint-ui --save

项目打包:（会多出来一个dist的文件夹）

npm run build 


react命令:

搭环境:
npm init -y
cnpm i react --save    核心包 (umd)
cnpm i react-dom --save  dom 操作的包(umd目录下)
cnpm i babel-standalone --save  babel

安装脚手架:

cnpm i create-react-app -g

生成一个项目

create-react-app 项目名字

安装路由:

cnpm i react-router-dom --save

安装fetch-jsonp

cnpm i fetch-jsonp

安装redux
cnpm i redux --save
